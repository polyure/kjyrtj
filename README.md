# KJYR-TJ
A Telegram bot which counts TJ (tänään jäljellä) for Kumpulan Järjestöjen Yhteinen Risteily cruise. Uses [python-telegram-bot wrapper.](https://python-telegram-bot.org/)

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/kjyrtj.git
    ```
2. Navigate to the project directory:
    ```bash
    cd kjyrtj
    ```
3. Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage
1. Set up your Telegram bot by following the instructions [here](https://core.telegram.org/bots#3-how-do-i-create-a-bot).
2. Create a `.env` file in the project directory and add your bot token:
    ```
    BOT_TOKEN=your_bot_token_here
    ```
3. Run the bot:
    ```bash
    python kjyrtj.py
    ```
